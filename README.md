# NLP Projects using Hugging Face

Natural Language Processing projects built using
Hugging Face Transformers library. These projects
cover text classification and emotion detection
using pre-trained transformer models.

## Projects

### 1. Text Classification using Hugging Face
- Used pre-trained transformer model for text classification
- Classified text data into predefined categories
- Fine-tuned Hugging Face pipeline for custom dataset
- Tools: Python, Hugging Face Transformers, Pandas, Google Colab

### 2. Emotion Detection and Verification
- Built emotion prediction model using Hugging Face
- Verified and validated emotion predictions on test data
- Classified text into emotion categories like happy,
  sad, angry, fear, surprise and neutral
- Tools: Python, Hugging Face Transformers, Pandas, Google Colab
